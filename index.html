    Google-API-Key

    Lokal ist der Key sicher. Einrichtung in der Google Cloud Console: Projekt anlegen → Maps JavaScript API und Places API (New) aktivieren → API-Key erstellen → Abrechnung aktivieren. Tipp: Key auf Referrer http://localhost:8000/* beschränken und Budget-Limit setzen. Ohne Key läuft nur OSM.

    Über Abseits

    Abseits hilft dir, Unterkünfte und Ziele abseits der großen Plattformen zu finden — auf Basis von OpenStreetMap und Google. Keine Anmeldung, kein Konto, nichts wird auf Servern gespeichert.

    So funktioniert's

    • Schlafen: Ort eingeben oder Standort nutzen — Abseits zeigt Pensionen, Gästehäuser, Ferienwohnungen, Hütten & Campingplätze. Über Filter Quellen, Typen sowie Plattform-/Ketten-Häuser steuern.
    • Entdecken: spannende Ziele in der Umgebung — Aussichtspunkte, Gipfel, Wasserfälle, Burgen, Badestellen, Picknick- & Feuerstellen. Filter wie „ohne Eintritt" und „naturnah".
    • Merkliste: Häuser & Ziele mit ★ merken — bleiben offline gespeichert, als GPX exportierbar oder teilbar.
    • Route & GPS: pro Eintrag direkt zur Navigation (Apple/Google Maps) oder Koordinaten kopieren — z. B. für Komoot.

    Als App installieren

    Am iPhone in Safari öffnen → Teilen-Symbol → „Zum Home-Bildschirm". Danach startet Abseits als eigenes App-Icon im Vollbild — ohne Browser-Leiste und auch offline nutzbar.
