Google-API-Key

Lokal ist der Key sicher (niemand außer dir sieht ihn). Einrichtung in der Google Cloud Console: Projekt anlegen → Maps JavaScript API und Places API (New) aktivieren → API-Key erstellen → Abrechnung aktivieren. Tipp: Key auf Referrer http://localhost:8000/* beschränken und ein Budget-Limit setzen. Ohne Key läuft das Tool nur mit OSM.

Gib oben einen Ort ein. Das Tool durchsucht den sichtbaren Kartenbereich nach Pensionen, Gästehäusern, Ferienwohnungen & Co.
